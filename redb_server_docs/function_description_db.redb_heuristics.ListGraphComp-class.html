Class ListGraphComp

source code

Comparing ListGraph heuristic. A ListGraph object is actually a List of Graphs, and Graph is a list of edges. ListGraph comparison is as follows:

  1. offset=0, start comparison.
  2. grade the injection of the smaller graph into the bigger graph (from the offset we got in 1).
  3. goto (2), while increasing the offset until it reaches len(bigger_graph)-len(smaller_graph)
  4. returning maximum grade received.
